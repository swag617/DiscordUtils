Minecraft Plugin  ·  Paper 1.20+
DiscordUtils
Bridge your Minecraft server and Discord with chat sync, account linking, item previews, and auction logging.
Paper 1.20+ Java 21 JDA LuckPerms
Features
💬
Chat Bridge
Minecraft chat relayed to Discord; Discord messages broadcast in-game. Supports MiniMessage, hex colors, role display, and Discord markdown conversion.
🔗
Account Linking
Players link their Minecraft account to Discord via OAuth2. LuckPerms rank automatically synced as a Discord role on every join.
🖼️
Item Display
Typing [item] in chat renders a tooltip image of the held item and attaches it to the Discord message as an embed.
🏷️
Auction House Logging
New listings, sales, and removals from zAuctionHouse posted to a Discord channel as rich embeds with item info and pricing.
More
🤖
Join / Leave Embeds
Player join and leave events posted to Discord with their Minecraft avatar.
💤
AFK Detection
Works with CMI or any plugin that broadcasts AFK status messages to the server.
📡
Server Message Relay
Forward plugin broadcasts like mini-game events to Discord by prefix filter.
Compatibility
Paper 1.20+ Spigot 1.20+ Java 21 Vault + LuckPerms
